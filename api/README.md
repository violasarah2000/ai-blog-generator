# Postman Collection
#
# This directory contains API testing and documentation files.
#
# Contents:
# - postman_collection.json - Postman collection for testing all endpoints
#
# Usage:
#
# 1. Import into Postman:
#    - Open Postman
#    - Click "Import"
#    - Select postman_collection.json
#
# 2. Run requests:
#    - Ensure Flask app is running on localhost:5000
#    - Execute requests from the collection
#
# 3. View API documentation:
#    - Use Postman's built-in documentation viewer
#    - Or see docs/guides/API.md for manual documentation

See postman_collection.json for the complete API specification.
